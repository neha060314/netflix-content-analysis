# Netflix Content Analysis

## Project Overview

This project analyzes the Netflix titles dataset using Python and data analysis techniques.  
The objective is to explore trends, patterns, and insights related to Netflix movies and TV shows, such as content distribution, release trends, ratings, and categories.


---

## Objectives

- Understand the dataset structure  
- Perform data cleaning and preprocessing  
- Conduct exploratory data analysis (EDA)  
- Analyze trends across content types and years  
- Visualize insights using charts  

---

## Dataset Description

The dataset contains metadata about Netflix titles with the following features:

- **show_id** – Unique identifier for each title  
- **type** – Movie or TV Show  
- **title** – Name of the content  
- **director** – Director of the title (if available)  
- **cast** – Actors/actresses featured  
- **country** – Country of production  
- **date_added** – Date content was added to Netflix  
- **release_year** – Original release year  
- **rating** – Content maturity rating  
- **duration** – Movie length or number of seasons  
- **listed_in** – Genre / category classification  
- **description** – Brief summary of the content
